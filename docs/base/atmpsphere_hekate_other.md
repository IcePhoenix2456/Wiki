# Глоссарий

!!! note "Что стоит прочитать?"
    На данной странице вы можете найти множество терминов, которые будут полезны и даже необходимы для использованния прошитой консоли. Если вы новичок, то прочитайте **как минимум** раздел "[Должен знать каждый](atmpsphere_hekate_other.md#dolzhen-znat-kazhdyi)", но лучше всего прочитать всю статью, что получить наилучшее понимание.

## Должен знать каждый
- HOS(Horizon OS) - это операционная система консоли, которая во многом основана на openBSD
- Atmosphere(Атмосфера) - это CFW c открытым исходным кодом, которая разрабатывается *SciresM*. Атмосфера не самодостаточная CFW, поэтому она работает поверх HOS, как надстройка заменяя некоторые компоненты HOS на свои. Разрабатывалась автором для запуска Homebrew и модов. По умолчанию не позволяет запускать игры установленные неофициальным способом.
- Hekate(Геката) - это кастомный загрузчик для прошитой консоли с открытым исходным кодом, который разрабатывается CTCaer. Позволяет загружать систему различными способами, управлять Эмунандом, а так же имеет множество других полезных функций. Если запускать систему через Hekate, то фьюзы не будут сжигаться.
- sysMMC(Сиснанд) - называют систему запущенную на внутренней памяти консоли(eMMC). Зачастую Сиснанд используют для игр установленных с Eshop и с картриджей. Если у вас не создан Эмунанд, то в Ultra-NX система при запуске Атмосферы будет грузиться с Сиснанда.
- emuMMC(Эмунанд) - это копия системы с Сиснанда на вашу карту памяти. Обычно Сиснанд используют для игр установленных неофициальным способом, например при помощи DBI, Tinfoil, Sphaira и др. Если создан Эмунанд, то в Ultra-NX система при запуске Атмосферы будет грузиться с Эмунанда. Более подробно про Эмунанд вы можете прочитать [здесь](sysnand_emummc.md)
- Ключи - это криптографические кода, которые хранятся в процессоре, нужны для расшифровки различных компонентов системы, а также её модификаций
- Fuses(Фьюзы) - физические нанопредохранители из специального массива в чипе, указывающий OFW на версию консоли. «Сжигаются» при завершении онлайн-обновления и всегда при запуске официальной прошивки, через загрузчик Nintendo на определенные версии. Если запускать систему, используя загрузчик Hekate, то фьюзы не сжигаются. По большей части нет смысла их беречь, так как все ревизии поддаются взлому.
- Методы запуска - определяют каким образом будет запущена система на вашей консоли. Некоторые методы позволяют вам запускать систему для запуска игр установленных неофициальным способом, например DBI. Другие методы позволяют запускать систему так, что можно было играть в игры скачанные из eshop, а также пользоваться онлайн функционалом консоли. Рекомендуется прочитать более подробную [статью](boot_methods.md) про методы запуска.
- Homebrew - это сторонние программы разного назначения, написанные энтузиастами без официальной поддержки Nintendo, могут быть запущены только на прошитой консоли.
- LFS mods(LFS моды) - это кастомная замена оригинальной файловой системы игр Switch RomFS и механизм, предназначенный для внесения прозрачных модификаций в игры «на лету», без необходимости их пересборки и переустановки; например, для использования русификаций, модов на графику, музыку. Установлен и активен в CFW по-умолчанию.
- Erista - это обозначение ранней ревизии чипов Tegra X1, которые установлены только в Nintendo Switch первой ревизии. Часть чипов имеет программную уязвимость, что позволяет прошить вашу консоль без использования мод-чипов. Проверить вашу консоль на программную уязвимость можно на [сайте](https://ismyswitchpatched.com/) введя ваш серийный номер консоли. Консоль также можно прошить с помощью мод-чипа.
- Mariko - это обозначение актуальной ревизии чипов Tegra X1, которые установлены в Rev2/Lite/OLED. Данная ревизия создана по более современному техпроцессу, из-за чего они потребляют меньше энергии и вследствие этого меньше греются. Данная лучше разгоняется. Консоли с данным процессором не имеют известных программных уязвимостей. Прошиваются только при помощи мод-чипов.

## Система
Термины относящиеся к работе операционной системе консоли, а также к её загрузчику

Рекомендуется прочитать более подробную [статью](boot_methods.md) про методы запуска.
- Sig-Patches и Sys-Patch - позволяют Атмосфере запускать игры установленные неофициальным способом. Sig-Patches ставятся, как патчи атмосферы, а Sys-Patch работает как системный модуль.
- Tickets(Тикеты, encrypted title key) - это специальным образом зашифрованная уникальная информация о правах запуска на контент игры, которая устанавливается в систему при инсталляции игры / игрового обновления / DLC
- Сертификаты картриджа - это зашифрованная информация, которая позволяет запускать игры с картриджа, напоминает тикеты, но не хранится на картридже
- PRODINFO - раздел в котором хранится уникальная информация необходимая для запуска HOS
- System Saves(системные сохранения) - сохранения, которые находятся на специальном разделе, хранят различные данные необходимые для работы системы, например профиль игрока, время проведённое в играх, язык системы и т.д.

- LayeredFS mods(LFS моды) - кастомная замена оригинальной файловой системы игр Switch RomFS и механизм, предназначенный для внесения прозрачных модификаций в игры «на лету», без необходимости их пересборки и переустановки; например, для использования русификаций, модов на графику, музыку. Установлен и активен в CFW по-умолчанию.
- NSP(Nintendo Submission Package) - контейнер с файлами игр в виде NCA из Nintendo eshop
- NSZ(Nintendo Submission Package Zipped) - Файлы NSZ функционально идентичны файлам NSP. Их единственное отличие - файлы внутри контейнера сжаты с помощью алгоритма zstd который так же используется в архивах 7-zip(.7z).
- XCI(NX Cart Image) - формат контейнеров содержащих структуру картриджей Nintendo Switch
- XCZ(NX Cart Image Zipped) - файлы XCZ функционально идентичны файлам XCI. Так же сжаты как и NSZ.
- NCA(Nintendo Content Archive) -  формат файлов-архивов которые могут быть конвертированы в NSP или XCI. Обычно являют собой множество архивов которые могут быть конвертированы в один контейнер. Все исходные данные NCA зашифрованы.
- NCZ(Nintendo Content Archive Zipped) - формат файлов аналогичный NCA единственное отличие - они сжаты, как и NSZ
