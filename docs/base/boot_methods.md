# Описание методов запуска

## Что такое методы запуска
Методы запуска определяют каким образом будет запущена система на вашей консоли. Некоторые методы позволяют вам запускать систему для запуска игр установленных неофициальным способом, например DBI. Другие методы позволяют запускать систему так, что можно было играть в игры скачанные из eshop, а также пользоваться онлайн функционалом консоли. Для лучшего понимания рекомендуется прочитать [статью о работе Сиснанда и Эмунанда](sysnand_emummc.md).

## Методы из вкладки Launch
Если вы зайдёте в Hekate и перейдёте во вкладку launch, то вы увидите два метода запуска. Оба запускают атмосферу преимущественно с Эмунанда(если его нет или он не активирован, то с Сиснанда). Данные методы запуска используют параметры запуска, чтобы идентифицирующие данные консоли были скрыты. Если идентифицирующие данные консоли скрыты, то консоль не может связаться с серверами Nintendo, это значит, что вы можете ставить игры установленные неофициальным способом и консоль не сможет связаться с серверами Nintendo, чтобы сообщить об аномальных действиях.

- Atmosphere - запуск системы с использованием лоадера switchcraft, что позволяет использовать разгон через Ultra Tuner
- Safe Mode - запуск системы без switchcraft. Данный метод нужен в случае, если вы поставили значения разгона или андервольтинга, которые ваша консоль не тянет, чтобы поставить значения с которыми консоль нормально запуститься. Также вы можете использоваться данный метод, если вам по какой-то другой причине нужно запустить консоль без switchcraft.

## Методы из вкладки More Configs
Если вы перейдёте во вкладку More Configs, то вы увидите три метода запуска системы. Все методы запускаются поверх Сисинанда и не скрывают идентифицирующие данные. Это позволяет вам пользоваться лицензионным функционалом консоли. Часто методы запуска из этой вкладки называют Стоком.

!!! danger "Я ставил игры неофициальным способом, находясь в Сисинанде"
    Если вы ставили игры неофициальным способом, находясь в Сиснанде, то настоятельно рекомендуется сделать [очистку Сиснанда от следов неофициально установленных программ](../useful_guides/sysnand_wipe.md) и после этого [создать Эмунанд](../ultra_wiki/backup_emuMMC.md/#emunand-emunand), иначе при входе в систему без скрытия идентифицирующих данных вы получите бан через некоторое время.

- Semi-Stock - данный метод запуска представляет собой атмосферу с switcraft запущенную поверх Сиснанда. В основном используется для разгона лицензионных игр, а также для использования некоторых системных модулей, например для подключения стороних геймпадов. Будьте аккуратны здесь и не создавайте ярылков для homebrew, а также установку игр неофициальным способом, например через DBI, иначе вы получите [бан](ban.md).
- Semi-Safe - то же самое, что и Semi-Stock, но без switchcraft
- Full Stock - метод запуска, который запускает чистую HOS на Сиснанде при помощи загрузчика Hekate.

## Другие методы запуска
Также существуют другие методы для запуска чистой HOS на Сиснанде. В этом случае используется стандартный загрузчик Nintendo, из-за этого при запуске будут сжигаться фьюзы до количества установленных в HOS. Данные способы не рекомендуется использовать, если система нормально запускается через методы из вкладки More Configs

- Если у вас консоль с чипом, то вы можете включить консоль с зажатием двух кнопок громкости(проверенно на чипах консолях с чипом Picofly)
- Также вы можете запустить систему через Hekate, выбрав пункт Reboot - OFW
