# Описание методов запуска

## Что такое методы запуска
Методы запуска определяют, каким образом будет запущена система на вашей консоли. Некоторые методы позволяют вам запускать систему для запуска игр, установленных неофициальным способом, например через DBI. Другие методы позволяют запускать систему так, чтобы можно было играть в игры, скачанные из eShop, а также пользоваться онлайн-функционалом консоли. Для лучшего понимания рекомендуется прочитать [статью о работе Сиснанда и Эмунанда](sysnand_emummc.md).

## Методы из вкладки Launch
Если вы зайдёте в Hekate и перейдёте во вкладку Launch, то увидите два метода запуска. Оба запускают Атмосферу преимущественно с Эмунанда (если его нет или он не активирован, то с Сиснанда). Данные методы используют параметры запуска, позволяющие скрыть идентифицирующие данные консоли. Это значит, что вы можете ставить игры, установленные неофициальным способом, и консоль не сможет связаться с серверами Nintendo, чтобы сообщить об аномальных действиях.

- Atmosphere - запуск системы с использованием лоадера Switchcraft, что позволяет использовать разгон через Ultra Tuner.
- Safe Mode - Этот метод необходим в случаях, когда установленные параметры разгона или андервольтинга приводят к сбою. Он позволяет загрузить консоль с безопасными параметрами и исправить настройки.

## Методы из вкладки More Configs
Если вы перейдёте во вкладку More Configs, то увидите три метода запуска системы. Все они запускаются поверх Сиснанда и не скрывают идентифицирующие данные. Это позволяет вам пользоваться лицензионным функционалом консоли. Часто методы запуска из этой вкладки называют Стоком.

!!! danger "Я ставил игры неофициальным способом, находясь в Сисинанде"
    Если вы ставили игры неофициальным способом, находясь в Сиснанде, то настоятельно рекомендуется сделать [очистку Сиснанда от следов неофициально установленных программ](../useful_guides/sysnand_wipe.md) и после этого [создать Эмунанд](../ultra_wiki/backup_emuMMC.md/#emunand-emunand), иначе при входе в систему без скрытия идентифицирующих данных вы получите бан через некоторое время.

- Semi-Stock - данный метод запуска представляет собой Атмосферу со Switchcraft, запущенную на Сиснанде. В основном используется для разгона лицензионных игр, а также для использования некоторых системных модулей, например для подключения сторонних геймпадов. Будьте аккуратны и не создавайте ярлыки для homebrew, а также не устанавливайте игры неофициальным способом (например, через DBI), иначе вы получите бан.
- Semi-Safe - то же самое, что и Semi-Stock, но без Switchcraft.
- Full Stock - метод запуска, который загружает чистую HOS на Сиснанде при помощи загрузчика Hekate.

## Другие методы запуска
Также существуют другие методы для запуска чистой HOS на Сиснанде. В этом случае используется стандартный загрузчик Nintendo, из-за чего при запуске определённых версий HOS будут «сжигаться» фьюзы до количества, установленного в ОС. Данные способы не рекомендуется использовать, если система нормально запускается через методы из вкладки More Configs.

- Если у вас консоль с чипом, то вы можете включить консоль с зажатием двух кнопок громкости (проверено на чипах Picofly).
- Также вы можете запустить систему через Hekate, выбрав пункт **Reboot** -> **OFW**.
